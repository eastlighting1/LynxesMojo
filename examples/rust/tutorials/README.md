# Rust Tutorial Examples

These files are the shortest ordered path through the Rust example surface.

- `01_read_gf.rs` parses a shared `.gf` file and builds a `GraphFrame`.
- `02_lazy_collect.rs` builds a lazy traversal and then collects it.
- `03_first_algorithm.rs` runs one eager graph algorithm workflow on the shared sample graph.

They are intentionally compact. If a Rust example starts turning into a long narrative, it probably belongs in the guides rather than here.
