# Rust Examples

The Rust examples are kept small and tutorial-shaped. They are not trying to be exhaustive API showcases. Their job is to show the same broad progression as the Python tutorial lane while still looking like normal Rust code.

The current tutorial lane is:

- `tutorials/01_read_gf.rs`
- `tutorials/02_lazy_collect.rs`
- `tutorials/03_first_algorithm.rs`

All of these examples assume you are running from a repository checkout and can access the shared graph files under `examples/data`.
