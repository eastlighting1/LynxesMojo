# Python Recipe Examples

These examples are task-oriented rather than sequential. You do not need to read them in order.

- `pagerank.py` shows a minimal PageRank run and the shape of the returned node frame.
- `community_detection.py` shows a community-detection call on the richer shared example graph.
- `io_roundtrip.py` shows a `.gfb` round-trip without leaving generated files behind in the repository.

They are still examples rather than cookbook pages. The code should stay compact enough that someone can copy it into their own script with only minor adaptation.
