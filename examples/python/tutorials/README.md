# Python Tutorial Examples

These files are the shortest ordered path through the Python surface.

- `01_read_and_inspect.py` starts with loading a graph and checking what is in it.
- `02_lazy_expand.py` introduces the lazy traversal surface and the `collect()` boundary.
- `03_first_algorithm.py` shows one eager algorithm call without turning the file into a broad algorithm survey.

They are intentionally small. If an example starts trying to teach too many separate tasks at once, it should move into the guides or be split into a recipe.
