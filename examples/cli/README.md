# CLI Examples

These files are command-oriented examples for the terminal surface. They are not reference pages, so they should stay concrete and example-first.

- `inspect.md` should show a real graph path and the basic inspection command.
- `query.md` should show a seed, a traversal shape, and what to check in the result.
- `convert.md` should show a conversion path and a quick validation step afterward.

The goal is that someone can copy one command block, run it, and know what success looks like without reading a longer guide first.
